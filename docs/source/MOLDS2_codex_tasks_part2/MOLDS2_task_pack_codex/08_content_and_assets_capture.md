# MOLDS2 — Task 08 — Content and Assets Capture

Собери все необходимые публичные ассеты и контентные элементы для точного воспроизведения выбранных страниц: изображения, иконки, логотипы, тексты, служебные подписи, навигационные labels и прочие видимые материалы. Нужна аккуратная организация ассетов внутри проекта.

## Что должно быть на выходе
- структурированный набор ассетов;
- отсутствие битых ссылок и потерянных элементов;
- корректная локальная работа воспроизводимых страниц.

## Ограничения
- не заменять ассеты произвольными аналогами;
- не генерировать новые тексты вместо существующих;
- не смешивать разные версии контента.
