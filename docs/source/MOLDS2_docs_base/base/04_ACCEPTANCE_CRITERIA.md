# MOLDS2 — критерии приемки первого этапа

## Обязательные результаты
Проект считается собранным на первом этапе, если выполнено следующее:

### 1. Стартовая инфраструктура готова
- создан Git-репозиторий проекта MOLDS2;
- используется Node.js-стек;
- проект может быть собран и задеплоен на Cloudflare Pages.

### 2. Главная страница воспроизведена
- повторена структура блоков;
- повторена визуальная композиция;
- повторены ключевые тексты, элементы интерфейса и навигация;
- нет самовольных отклонений по стилю.

### 3. Воспроизведены страницы первого контура
- перенесены страницы, доступные с главной через хедер, футер и явные разделы;
- сохранена связность переходов между ними;
- повторена логика их базового представления.

### 4. Воспроизведены блоговые страницы
- сделано около 10 внутренних блоговых страниц;
- повторен их шаблон;
- подтверждена возможность переноса контентного типа страниц.

### 5. Есть доказательство системности
- выделены повторяющиеся шаблоны;
- структура не собрана как случайный набор несвязанных HTML-копий;
- проект пригоден к масштабированию.

## Что считается провалом
Проект не принимается на этапе, если:
- выполнен редизайн вместо воспроизведения;
- сохранена только “примерная похожесть”;
- нет связной навигации;
- внутренние страницы собраны бессистемно;
- отсутствует готовность к Cloudflare Pages;
- команда ушла в перенос всего сайта вместо согласованного scope.
