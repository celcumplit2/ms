# MOLDS2 — Task 03 — Design Capture

Сними визуальную систему исходного сайта и подготовь точное воспроизведение внешнего слоя. Нужно зафиксировать layout, сетки, отступы, размеры блоков, типографику, цвета, иерархию секций, состояние навигации, хедер, футер и общие UI-паттерны.

## Что должно быть на выходе
- описание ключевых визуальных правил;
- перечень reusable-блоков;
- понимание того, что должно быть воспроизведено один в один;
- набор reference-материалов для точной сборки страниц.

## Ограничения
- никакого редизайна;
- не упрощать спорные элементы только потому, что так удобнее;
- не заменять структуру на “похожую”.
