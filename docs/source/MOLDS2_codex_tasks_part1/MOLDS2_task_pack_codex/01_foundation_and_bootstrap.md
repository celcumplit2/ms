# MOLDS2 — Task 01 — Foundation and Bootstrap

Создай новый проект MOLDS2 на том же технологическом контуре, что и VIX/Warfair: Node.js, Git, Cloudflare Pages. Подними чистый рабочий репозиторий, базовую структуру проекта, конфигурации сборки и деплоя. Ничего лишнего не придумывай: нужна надежная основа под дальнейшее воспроизведение сайта.

## Что должно быть на выходе
- рабочий проект, который запускается локально;
- репозиторий с понятной структурой;
- сборка без ошибок;
- готовность к деплою на Cloudflare Pages;
- краткий README с командами запуска, билда и деплоя.

## Ограничения
- не заниматься массовым переносом страниц;
- не менять визуальную концепцию;
- не добавлять лишние продуктовые идеи.
