# MOLDS2 — Task 04 — Header, Footer, Navigation

Воссоздай хедер, футер и основную навигацию максимально близко к оригиналу. Это включает структуру меню, вложенность, служебные ссылки, визуальные состояния, расположение элементов и общую композицию.

## Что должно быть на выходе
- хедер, повторяющий исходный сайт;
- футер, повторяющий исходный сайт;
- рабочая маршрутизация по воспроизводимым страницам;
- одинаковое визуальное поведение основных навигационных элементов.

## Ограничения
- не переписывать IA по-своему;
- не сокращать состав ссылок без причины;
- не менять логику навигации ради “улучшения UX”.
