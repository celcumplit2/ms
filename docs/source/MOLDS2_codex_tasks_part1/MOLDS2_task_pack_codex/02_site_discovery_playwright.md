# MOLDS2 — Task 02 — Site Discovery via Playwright

Используя ссылку на главную страницу исходного сайта, пройди сайт через Playwright и собери карту доступной публичной структуры первого уровня. Нужно зафиксировать главную страницу, все страницы, доступные из хедера, футера и основных навигационных элементов, а также выделить типовые шаблоны внутренних страниц.

## Что должно быть на выходе
- список URL, доступных с главной;
- описание типов страниц;
- список повторяющихся блоков и навигационных паттернов;
- перечень кандидатов на 10 блоговых страниц для демонстрационного переноса.

## Ограничения
- не сканировать весь сайт на 120k страниц;
- не делать массовый парсинг всего блога;
- не пытаться улучшать структуру сайта.
