# MOLDS2 — Task 07 — Blog Sample Pages

Воссоздай около 10 блоговых страниц как демонстрацию возможности переноса контентных внутренних страниц. Выбери реальные страницы блога и повтори их шаблон, структуру, компоновку текста, медиа-блоки, хлебные крошки, сайдбарные элементы и связанные секции настолько точно, насколько это возможно.

## Что должно быть на выходе
- 10 воспроизведенных блоговых страниц;
- единый шаблон, повторяющий оригинал;
- готовность использовать этот подход для дальнейшего масштабирования.

## Ограничения
- не переносить весь блог;
- не переписывать статьи;
- не упрощать шаблон до абстрактного “blog layout”.
