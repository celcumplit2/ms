# MOLDS2 — Task 06 — Static Pages Reproduction

Воспроизведи все основные страницы, доступные с главной через хедер, футер и базовую навигацию. Это должны быть реальные страницы существующего сайта, перенесенные без творческих отклонений.

## Что должно быть на выходе
- набор внутренних страниц первого уровня;
- единая визуальная система между всеми страницами;
- сохранение структуры и логики исходного сайта.

## Ограничения
- не расширять объем дальше страниц первого уровня;
- не изобретать недостающий контент;
- не менять названия, порядок и смысл блоков.
