# MOLDS2 — Task 09 — QA and Visual Parity

Проведи проверку того, насколько собранный MOLDS2 визуально и структурно соответствует исходному сайту. Нужна фиксация расхождений, повторная доводка спорных мест и финальная сверка по главным страницам и выбранным блоговым страницам.

## Что должно быть на выходе
- список найденных расхождений;
- исправления критичных несоответствий;
- подтверждение, что страницы выглядят максимально близко к оригиналу.

## Ограничения
- не подменять QA субъективными оправданиями;
- не оставлять явные отличия без фиксации;
- не считать “похоже” достаточным критерием.
