# MOLDS2 — ожидаемые артефакты проекта

## Обязательные артефакты

### 1. Карта переноса
Документ со списком:
- главной страницы;
- страниц первого уровня из хедера/футера/основной навигации;
- 10 выбранных блоговых страниц.

### 2. Карта шаблонов
Отдельный документ или раздел, где зафиксировано:
- какие page types были обнаружены;
- какие блоки повторяются;
- какие шаблоны нужно поддерживать.

### 3. Репозиторий проекта
В репозитории должны быть:
- исходники проекта;
- базовая структура страниц;
- маршруты;
- компоненты/шаблоны;
- конфигурация сборки.

### 4. Локальная рабочая сборка
Проект должен запускаться локально и давать воспроизводимый результат.

### 5. Готовность к Cloudflare Pages
Нужна конфигурация и структура, позволяющая штатно деплоить проект.

### 6. QA-подтверждение
Нужна фиксация того, что:
- главная повторена;
- ключевые страницы повторены;
- блоговый шаблон перенесен;
- навигация не развалена.

## Дополнительные полезные артефакты
- скриншотные сверки;
- таблица URL → page type;
- список спорных мест, если такие обнаружатся в процессе переноса.
