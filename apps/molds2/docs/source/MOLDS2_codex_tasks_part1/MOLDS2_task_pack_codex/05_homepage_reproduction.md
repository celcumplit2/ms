# MOLDS2 — Task 05 — Homepage Reproduction

Полностью воспроизведи главную страницу исходного сайта. Нужна максимальная визуальная и структурная идентичность: секции, порядок блоков, контентные области, CTA, карточки, баннеры, декоративные паттерны, spacing и общая композиция.

## Что должно быть на выходе
- главная страница, визуально максимально близкая к оригиналу;
- адаптация под наш стек без внешних отличий;
- корректная сборка и рендер без поломанных блоков.

## Ограничения
- не придумывать новые секции;
- не улучшать копирайтинг;
- не менять композицию только потому, что “так чище”.
