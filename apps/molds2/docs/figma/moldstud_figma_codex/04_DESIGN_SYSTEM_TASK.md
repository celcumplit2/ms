# TASK — DESIGN SYSTEM EXTRACTION

Задача.
Выдели из Figma дизайн-систему, которую можно перевести в tokens, базовые стили и компоненты для новой версии Moldstud.

Что нужно получить.
Создай файл docs/figma/DESIGN_TOKENS.md и зафиксируй:
- palette / semantic colors;
- typography hierarchy;
- spacing scale;
- radius system;
- border styles;
- shadow/elevation rules;
- containers and grid widths;
- section spacing rules;
- icon style rules;
- motion/transition hints, если они понятны из макета.

Дополнительно.
Сформируй инженерное представление о том, как это должно жить в коде:
- что становится design token;
- что становится utility/primitive;
- что становится полноценным reusable component.

Требование.
Не описывай это как дизайнерский обзор. Описывай как основу для реализации в production-коде.
